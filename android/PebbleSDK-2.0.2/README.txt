Pebble SDK
==========

This package contains:

 * Pebble Software Development Kit (in /Pebble/)

   This development kit allows the development of watchface
   and watchapp for the Pebble connected smartwatch in C.

 * PebbleKit libraries for iOS and Android (in /PebbleKit-iOS and /PebbleKit-Android)

   PebbleKit is a library that enables communication between a mobile
   application and Pebble.

 * Reference documentations

 * Examples

## Installing Pebble SDK

Please refer to the documentation on our website: http://developer.getpebble.com/2/
